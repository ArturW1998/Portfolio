## Portfolio

Laravel and React

https://arturvoloshyn.neosvita.com/

## Requirements
- PHP 7.2
- MySQL
- Composer
- libpng

## Installation
1. `composer install` (or `php composer.phar install`)
2. Copy `.env.dist` as `.env`
3. Replace the database and mail connection strings in the `.env` file
4. `php artisan migrate`
5. `php artisan serve`
